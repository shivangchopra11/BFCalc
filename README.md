# BFCalc

----
## Scientific calculator for Android

> *A backport of the recent redesign of the default calculator app shipped with Android Lollipop. Can support both integral and decimal operands upto 10^18.*

1. Add
2. Subtract
3. Multiply
4. Divide
5. sin

7. tan
8. sinh
9. cosh
10. tanh
11. log
12. log10
13. Modulus
